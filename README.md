# Kanban Dashboard

### Steps to setup


#### 1. Clone the repo
`git clone git@github.com:sampath88/kanban-react.git`

#### 2. Install dependencies (recommended node version: v18.14.2)
`$ npm install`

> Note: Before running the app, please make sure backend is live
#### 3. Run App
`$ npm run dev`

#### App will be running at
`http://localhost:5173/`
